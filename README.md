# C2FNet
this repotory contains code of paper C2FNET: COARSE-TO-FINE GRAINED INTEGRATION NETWORK FOR MULTIMODAL SENTIMENT ANALYSIS(already submitted to ICASSP2026)
# prerequirement
install all requirements mentioned in requirements.txt, download mustard or other dataset in fold dataset 
# quick start
you can use this command to quickstart C2FNet:
python main_pipeline.py, you can change different args and dataset in main_pipeline.py, good luck
